Math Lab Example

Practical application is a great way of developing and assessing skills and conceptual understanding. Math labs which involve coding are perfect examples of practical applications. Consider the following code block taken from a lesson on one dimensional scaling transformations of quadratic functions in parametric form:

var clock = new THREE.Clock();
animate(){
  var t = clock.getElapsedTime() -60;
  var tSquared = Math.pow(t,2);
  sphere.position.x = xScaleFactor * t;
  sphere.position.y = yScaleFactor * tSquared + k;
}

The code shown above is used to move a 3d object along a path defined by quadratic parametric equations. Time is a common parameter seen in many textbook applications of quadratic equations in parametric form. Why not have the students apply their skills for real?
