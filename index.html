Overview

Practical application is a great way of developing and assessing skills and conceptual understanding. Consider the following code block taken from a lesson on one dimensional scaling transformations of quadratic functions in parametric form:

var clock = new THREE.Clock();
animate(){
clock.getDelta();
object3D.position.x = timeScaleFactor * clock.getElapsedTime();
object3D.position.y = Math.pow(clock.getElapsedTime(), 2);
}

The code shown above is used to move a 3d object along a path defined by quadratic parametric equations. Time is a common parameter seen in many textbook applications of quadratic equations in parametric form. Why not have the students apply their skills for real?
